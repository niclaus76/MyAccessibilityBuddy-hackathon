My Accessibility Buddy

AI tool for creating WCAG 2.2-compliant alt text
Usage: upload an image, optionally provide context, review the alternate text and save your alt text.

How to Use

  1. Select Processing Mode: Choose "Basic" for default settings or "Advanced" to customize providers and models
  2. Upload Image: Drag and drop or select an image file (JPG, PNG, GIF, WEBP, BMP, SVG, TIFF)
  3. Add Context (Optional): Upload a text file or type context information to help improve alt text accuracy
  4. Select Languages: Choose one or more languages for your alt text (hold Ctrl/Cmd for multiple)
  5. Choose Translation Mode: "Fast" translates once (recommended), "Accurate" generates fresh for each language
  6. Generate: Click "Generate Alt Text" and wait for processing
  7. Review & Edit: Review generated text, make edits if needed, and save

Processing Information

Processing Steps

  • Step 1 - Vision: Analyzes the image and generates a detailed description
  • Step 2 - Processing: Converts the description into WCAG 2.2-compliant alt text
  • Step 3 - Translation: Translates the alt text to the selected language(s)

Available Providers and Models

  • OpenAI: gpt-4o, gpt-5.1, gpt-5.2 (cloud-based, requires API key)
  • Claude: claude-sonnet-4, claude-opus-4, claude-3-5-haiku (cloud-based, requires API key)
  • ECB-LLM: gpt-4o, gpt-5.1 (internal ECB service, requires authentication)
  • Ollama: granite3.2-vision, llama3.2-vision, phi3, mistral (local models, free)

In Advanced mode, you can mix providers across steps (e.g., Claude for vision, OpenAI for processing) or use default settings from config.json.

Translation Modes

  • Fast Mode: Generates alt text once in the first language, then translates to other selected languages. Recommended for cost and speed efficiency.
  • Accurate Mode: Generates fresh alt text directly for each selected language using the AI model. More accurate but slower and more expensive.
