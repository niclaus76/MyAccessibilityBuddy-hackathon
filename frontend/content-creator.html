My Accessibility Buddy - Content Creator Tool

AI tool for creating WCAG 2.2-compliant alt text
Usage: upload an image, optionally provide context, review the alternate text and save your alt text.

How to Use

  1. Advanced processing mode: Leave off to use the default basic settings, or turn it on to customize providers, models, and translation mode
  2. Upload Image: Drag and drop or select an image file (JPG, PNG, GIF, WEBP, BMP, SVG, TIFF)
  3. Add Context (Optional): Upload a text file or type context information to help improve alt text accuracy
  4. Select Languages: Choose one or more languages. Use Up/Down arrow keys to navigate, Space to select/deselect (Ctrl/Cmd+Space for multi-select, Shift+arrows for range)
  5. Generate: Click "Generate" and wait for processing
  6. Review & Edit: Review generated text, make edits if needed, and save
